# Quantum Visual Lab

양자역학의 주요 개념을 한국어로 직관적이게 이해할 수 있도록 돕는 인터랙티브 시각화 실험실입니다.

## 목적
- 파동함수, 확률밀도, 파동패킷, 위상/군속도, 양자 터널링 등 양자역학 개념의 시각화
- 복잡한 수식을 인터랙티브하게 체험하며 직관적 이해력 향상

## 기술 스택
- **프레임워크/빌드**: Vite + Vanilla JavaScript
- **시각화 라이브러리**: Plotly.js
- **수식 렌더링**: KaTeX
- **배포**: GitHub Pages (예정)

## 로컬 실행 방법

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```
